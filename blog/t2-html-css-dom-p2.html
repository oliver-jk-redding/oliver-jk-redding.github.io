What is grid based design?

12 December 2015

Grid based design is laying out content of a webpage according to a grid usually 12 columns across. This is effective because it means you can arrange content evenly in columns of 1, 2, 3, 4, 6, and 12.

What's all the hype about responsive webpages?

12 December 2015

Responsive web pages are pages whose content responds to the size of the screen. This means when a screen is small or large the content on the page responds to arrange itself in a visaully pleasing way. This has become especially important in recent times with the advent of phones and tablets, meaning people could be viewing a webpage on a variety of different screen sizes at any one time.

What is semantic structure?

12 December 2015

It is not hard for a computer to read HTML. So long as the syntax is adhered to, the computer will be able to read it, even if the code is strung out in one long line. However it is far more beneficial to write code that a human can understand as well. To assist in this, it is common to use semantic structure. This includes using tags with more specific meanings e.g. 'nav' even though it is syntactically identical to 'div' because it carries more meaning for a human trying to read and understand the code. Semantic structure therefore is using code that carries meaning from a humna point of view to make it easier to understand.

Using fonts

12 December 2015

Udacity taught me how to find and include new and interesting fonts in my webpages. I have read however that not all browsers can read all fonts and that it is better to stick to a few common ones or at least include common fonts as secondary styles for better browser compatibility. This is something I would like to investigate further.

On that note, there seems to be a lot of time given to making things compatible with old browsers. Is this really still necessary? The majority of viewers will have modern browsers. Should we care about those who don't? Is it time they caught up with the times? Should we be catering to their archaism?
